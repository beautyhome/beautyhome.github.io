浴室，洗手间remodel

发表于 2018-03-07 | 分类于 projects | |
翻新楼房的洗手间、浴室，更换浴盆、贴磁砖，更换洗手池柜子、台面，更换洗手池上方的灯。

都说美国的房子耐用，上世纪6~70年代的房子比比皆是。事实上，木结构的房子，需要做比较多的维护，才能维持好的状态，在经历多年，内部仍然舒适宜居。

最近，俺们几个，在一乡邻的信任下，帮其翻新了一下老房子的卫生间跟洗手台。在这里跟大家分享一下翻新的过程。

翻新浴池

拆除磁砖

浴池在第一次安装的时候，房子的结构刚完成，drywall，门，门框很可能都没安装，那时安装，是很容易的事。但是后期要换新的，就难了。首先要拆除上方的磁砖。

磁砖拆掉后，本以为浴池就可以取出来了。但是，周围的drywall，门框仍然挡着，没办法，接着拆，一直拆到能取出浴池为止。

安装新浴池

清理干净现场，保护好水管，就可以安装新浴池了。安装步骤，跟取出旧的刚好反过来。安装好下水管，overflow水管。

安装新的水泥板

水泥板放在浴池上方的墙上，方便安装新磁砖。水泥板之间接缝如果不是刚好在stud上，要用木板固定，防止安装磁砖的时候，背部不稳。

贴磁砖

贴磁砖，大家可以参考下老周的公众号文章：

  • 自己动手贴地砖

贴完磁砖，等底下的motor干了，就可以在缝隙里填充grout。Grout干了可以抹层sealer，角上的缝隙抹上caulk。

更换洗手池

移除

洗手池的柜子，一般都是压花板，容易吸潮气，发霉。

这个移除一般比较容易。

移除后，发现不知啥动物在里面地上搬运了一大堆东西。

在不标准的空间里安装一个标准的柜子

如果尺寸标准，那安装费就很便宜。不幸的，是定制尺寸，那就悲剧了。

像这个位置，标准尺寸柜子就偏小，而且右侧的抽屉会挡住水管，所以只能偏右边摆放。为了美观，我们在左侧放置了一块侧板。

接水管

固定好新的柜子，把水管接好。洗手池的排水管按照新code，是1-1/4’’的直径，但是这个老房子，是1-1/2’’的。而店里，已经买不到1-1/2’’水管的水龙头了。这时候需要一个1-1/4’’的p trap washer，这个washer一般店里不单独卖，不过很多1-1/2’’的p trap kit里会附带一个。

换灯

换灯，最烦人的事情是拆下旧灯后，墙上一个大洞，然后柒的颜色还不一样。

这就需要先把洞补好，把墙用compound抹平，然后上纹理，配柒，刷漆。大家可以参考我们之前的文章。

  • 维修墙内水管

安装好镜子，装上新的灯。灯的接线俺说一下步骤，大家小心前提下，也是可以自己动手的。

灯的线一般是14-2的Romex，15A，白色，里面一根裸铜线为地线，黑色为火线，白色为零线（跟中国不一样）。接线顺序为：地线->零线->火线。接线前一定要确认线是不带电的。
