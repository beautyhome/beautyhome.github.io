砖墙上开个出风口

发表于 2017-06-09 | 分类于 生活DIY | |
字数统计: 269

有不少朋友想在厨房装抽油烟机，却发现墙上没有洞，而且外面还是砖结构。哇，看起来好复杂，还是不要装油烟机了。

其实，打个洞并不复杂，记住要诀：胆大，心细。

不差钱的朋友，可以直接买个合适大小的 diamond core bit，省时省力。

不过，咱今天主要还是讲讲省钱的弄法。

首先，在内墙上，用stud finder定位一下墙里的木头，避开电插座。挖开一个小洞，看看墙后的情形。如果墙里没什么障碍，就可以开一个需要大小的圆洞。

从洞的中间用锤钻把后面的砖墙打个定位孔。然后到外墙上，用粗点的水泥钻头配合锤钻，钻出圆形的轮廓。

下面这步要心细了，用凿子小心的把洞敲开。

洞通了，试试管道合适不，根据需要修下洞的边缘。

把管道放进去，密封好。

最后，在外墙装上合适的vent cover。大功告成。
