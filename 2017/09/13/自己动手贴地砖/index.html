自己动手贴地砖

发表于 2017-09-13 | 分类于 生活DIY | |
地板会被淹，地砖不怕。

哈维之后，我有一些原来铺地板和地毯的朋友，不约而同决定扒了地毯地板之后，铺地砖。今天就介绍一下如何铺地砖。

首先，介绍工具

1.tile saw

切地砖，需要tile saw。 下图就是最基本的tile saw，和台锯结构类似，但是防水，所用的锯片是钻石锯片，正如俗话说的“没有金刚钻，别揽瓷器活”

2. 抹mortar的工具kit

类似于泥瓦匠的一套工具，最重要的就是Trowel。

3. grout trowel

4. 海绵块

5. 电钻

6. mortar mixer

其次，介绍材料

1. 地砖

自选花色大小

2. mortar

颜色自选

3. grout

颜色自选

4. grout sealer

有事后喷的，也有混在grout里用的

5. tile spacer

6. tile caulk

如果用在浴室，需要用silicon的，防水更好

7. 水桶

最后，介绍步骤

1. 底面清洁，取平

把地上的碎屑都清理掉，用一个长木板在地上刮一刮，标记出不平的底面，可以用一些mortar填平。这里不详细介绍

2. 混合用于地砖安装的mortar。

需要一个桶，先往桶里加一些水，然后倒入mortar粉，用电钻接上mortar mixer搅拌均匀。

两点要注意：A先加水，后加mortar。顺序反了会导致mortar难以搅拌均匀。B 尽量用插电的电钻，扭矩大一些，有些比较弱的电池电钻根本搅不动。

3. 将mortar抹在地上，再用有齿 的一面刮出沟槽

再将mortar抹在地砖反面。

把地砖贴到地面上。因为有沟槽存在，你有一定的高度调节空间，可以用拳头砸地砖的表面，把它砸平，或者用橡皮锤轻砸。

4.下一块。

用spacer隔出你需要的缝隙大小

5. 地砖贴到边角的时候，就需要切了。

使用tile saw，使用方法类似台锯，根据你的需要切出不同的形状。使用时，注意给锯片浇水降温。

6. 抹grout

地砖贴好24小时之后。注意！24小时之后！！！可以开始抹grout

地砖spacer隔出的缝要用grout填补。Grout加水混合，搅拌均匀，然后用grout trowel抹到缝隙里。

半小时之内，注意之内，不是之后！用湿的海绵块，把多余的grout擦去。可以多更换几次清水，擦的干净一些。

7. 封边

24小时之后，grout也干了，需要把底面和墙边的缝隙用caulk封住。Caulk的选择和房间的用途有关，如果是浴室长期接触水的部分，最好用silicon caulk，一般的位置用tile caulk就好

8. 封grout

大家都有经验，地砖缝容易脏，为了帮助地砖缝抵御脏东西侵蚀，最后还需要把grout seal一下，可以用喷雾剂把缝都喷一遍。

也有其他形式的seal，比如抹的，刷的，或者是直接混在grout使用的。不过，喷的最方便了。

好了，做到这一步，铺地砖就算完工了，可以入住了。!
