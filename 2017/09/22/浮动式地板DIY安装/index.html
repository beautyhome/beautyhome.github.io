浮动式地板DIY安装

发表于 2017-09-22 | 分类于 生活DIY | |

飓风之后，遭受淹水损失的家庭深刻体会到了选择合适的地面材料的重要性。上一期我介绍了最省事儿的地面材料－地砖－的安装。不过凡事有利有弊，地砖太硬，太凉，省事儿是省事儿，舒适性略显不足。显然木地板在舒适性上强过地砖不少，而在易用性上又远高于地毯，很多人肯定还是更青睐地板。

地板根据材质大致分为三类

  1. laminate。就是木渣混合胶水压制成的木片，表面加上一层高强度表面材料，模拟真实的木材花纹。这种地板价格低廉，表面强度高，耐磨。不过质地过硬，而且不耐水泡。
  2. engineered。这种地板用木片压制而成，结构类似于三合板，表面上再加一层真正的地板木头，比如樱桃木之类的。这种地板是纯木制成，强度好，有一定的耐水能力，使用感觉和实木相当。
  3. 实木地板。整块的木头制作的地板，适合有钱人。当然实木地板也分不同档次，分别针对小康之家，有钱人，土豪，大富豪。。。。

购买地板的时候，注意识别地板的材质，因为有些商家给地板取名的时候会走灰色地带，取个模棱两可的名字，让你误以为用更低的钱买了更高一级的地板。如果有这种明显的“便宜”可占，你就要怀疑你可能被骗了。一分钱一分货才是真理。

地板的安装方式也是多种多样。

  1. 实木地板一般采用nail down的方式，用钉子钉到龙骨，或者木质的sub floor上。
  2. 还有一种流行的安装方式是glue down，就是用胶水把地板贴到地面上。这种安装方式简便，坚固。不过需要使用大量胶水，另外也会在拆除时造成很大的麻烦，你会发现揭地板很难，去除残留的胶水更难。你需要使用一些专门的工具，并且花费大量的时间才能拆除旧的glue down地板。
  3. 浮动式（floating）安装。这种方式适用于四面带卡扣(不是卡槽)的地板。浮动式，顾名思义，地板是不固定到地面的，而是相当于拼接成一整块大地板放置于地面上。这种方式对安装的技术要求不高，拆除更换也很方便，是一种非常适合diy安装的技术。

本文将介绍一下技术3。

浮动式（floating）安装

设备要求

  1. miter saw （如果你有台锯，也可以没有miter saw，只是有miter saw会更方便）
  2. 台锯
  3. floor install kit
  4. 锤子
  5. 护膝
  6. 钉枪（可以没有，那就要自己敲钉子了。）

材料准备

  1. 地板。需要是带卡扣（clock lock）的地板，一般的laminate，engineered都有四面可锁死的卡扣，某些实木地板都会有。只带槽／舌的不行，扣不住。
  2. 防潮膜：可以是塑料膜，或者泡沫垫。
  3. quarter round
  4. T molding
  5. 少量胶水

安装过程

  1. 地面清理，取平。这里不做介绍
  2. 铺防潮膜
  3. 铺第一条地板。需要在地板的两端，以及靠墙的一面插入垫片，以留出空间，让地板可以因气候自由膨胀收缩。另外注意安装方向，舌头朝外，槽朝里。用以下手法锁定。为了保持一定的pattern，每一行地板的垂直接缝最好错开一尺以上，因此每一行地板的第一块可能要锯掉一定长度。

    左右卡扣的连接，可能需要借助塑料块敲击。
  4. 铺到右端，根据长度需要，截取合适的地板拼入.如果有一定角度，截出相应的角度。


    5.铺到门边，需要切掉门框下沿部分。
  5. 铺到对面墙，剩余宽度不一定等于一块地板的宽度，需要用台锯切出合适的宽度。
  6. 在墙边空间局促的部分，如果需要敲紧某些接缝，可以用floor install kit里的这个铁板

    装地板是一个无聊冗长的体力劳动，在你装好每个房间的地板之后，还剩下墙边的缝，地板与其它材质底面的接缝等需要装饰。
  7. 墙边的缝用quarter round覆盖。记得在拐角处要切出不同的角度来使拐角衔接自然，美观。

    Quarter round可以用钉枪钉到墙上。或者用锤子敲小钉子。用锤子时，最好在敲钉的位置用电钻钻穿一个小孔，不然你10有89会敲不进去。
    1. 接缝，比如地板和瓷砖的接缝，用T-molding覆盖。缝里抹上胶水，把T-molding贴进去就好。

      然后在t-molding上压上重物，保持24小时就好。

基本上，click－lock地板的浮动安装过程就是如此。很简单吧？就是一项体力劳动。
