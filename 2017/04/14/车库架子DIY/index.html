车库架子DIY

发表于 2017-04-14 | 分类于 生活DIY | |

有了房子后，家里的破铜烂铁就会越来越多，车库就会越来越乱。慢慢的车停不进去了，再然后，下脚的地方也没有了。然而，商店里琳琅满目的漂亮架子，价格让人望而却步。有一天，在某本杂志上看到了用木头做各种架子的idea，俺就在自己的车库实践了一下，效果还不错，堪称物美价廉。

做木头架子，当然得需要些工具切割、连接木头。主要的工具是：

俺还用到了台锯，更方便些，不过圆锯也可以代替。

首先，俺做了一个靠墙的架子，用来放堆放各种物品。这个架子用的是2x4‘’x8‘的松木。为了防止架子会倒，后面通过横木条订到墙上的stud上。

然后是车库顶棚的架子，可以放各种不常用的东西。用了2x4’’的松木跟一块4x8’的OSB板子。这个架子俺挂在了opener上方，这个地方比较隐蔽，不会让车库看起来很拥挤。


最后，俺又做了些墙上的挂架，用的是3/4’’x4x8’的plywood，按照4寸宽度，切成45度倾角的木条，沿着车库的边墙一路订上去。


然后用同样的斜切法做一堆各式各样的挂件，可以用来挂各种东西。

经过这一番折腾，车库里终于腾出地来可以下脚，停车了。
