老高修车记

发表于 2017-04-05 | 分类于 生活DIY | |

作为一名优秀的理工男青年，每每完成一个DIY工程，都让俺有一种莫名的成就感。难度越高，挑战越大，往往成就感就越强。伙伴们是不是“与我心有戚戚焉”？

车子虽然比房子要小很多，但是复杂程度可一点都不低。而且车子内部空间狭小，给维修增加了不少难度。今天俺就跟大家分享一下俺最近DIY修车的经历。

话说2017年的某一天，俺的车子忽然亮起了两个橘黄色的小灯，是这样色的。


这提示俺的车子ABS跟Track control功能都失效了。更为严重的是，俺开着开着，右前方轮胎居然被锁。这下子，必须立刻、马上维修了。打电话给dealer，说检查费200。好吧，俺还是先自己折腾一下，不行再说。

放狗了解了下，ABS系统是由ABS模块跟一些sensor组成，其中最主要，最容易出故障的就是4个wheel speed sensor。就先从它们开始。

WSS分为被动式跟主动式两种。老车上被动式较多，它的特点是自己本身随轮胎转动产生一个电压变化的正弦波形。


这种WSS的检测很简单，只要用万用表的AC档放在WSS的两个头上，转动轮胎，会看到一个变化的电压，轮胎越快，电压越大。

而主动式WSS就要复杂一些，它的工作需要一个12v的电压输入，其输出为频率变化的方形波。


这种WSS的故障诊断也就更复杂些。如果有示波器，可以直接测量输出波形。如果没有，那有个简单易行的办法，叫LED测试法。找一个220欧姆左右的电阻，一个12V的直流电源，一个小LED。把电源的一极接到Sensor上，另一极跟sensor另一极之间接电阻和LED。然后转动轮胎，如果WSS正常，那LED灯会有规律的闪烁。轮胎转得快，LED闪得也快。这个测试可能要试几次才能最后确认故障。或者，更直接的，如果不确定是不是WSS的故障，那直接就把它们全部换成新的，一般不会超过100块。不幸的是，俺一开始没有确定故障，又去了一趟dealer做诊断，结果是，除了俺已经刚更换过的1个是好的，剩下3个全坏了。俺心里直接一万头”草泥马“跑过。

好了，故障确定是WSS，剩下就是更换了。更换的过程大同小异，就是拆下轮胎，把旧的WSS插头拔开取出，装上新的。


换完后，俺的车又生龙活虎了。
